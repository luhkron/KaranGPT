Import Data from Spreadsheet

Upload an Excel (.xlsx, .xls) or CSV (.csv) file to populate the database. Make sure your file has columns that match the data type you are importing.

Required Columns for Drivers:

Your spreadsheet must contain the following columns: name, license_number. Optional columns are phone, email, and active.

Required Columns for Trucks:

Your spreadsheet must contain the following column: unit_number. Optional columns are make, model, rego, capacity_tonnes, and current_km.

Required Columns for Trailers:

Your spreadsheet must contain the following column: trailer_id. Optional columns are type, rego, and capacity_tonnes.

Required Columns for Workshop Jobs:

Your spreadsheet must contain the following column: description. You must also include either a truck_unit_number or a trailer_trailer_id to link the job to a vehicle. Optional columns are date_opened, date_closed, status, and cost.
